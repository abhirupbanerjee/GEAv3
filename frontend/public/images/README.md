# Images Directory

Please place the following images in this directory:

1. **grenada-coastal.jpg** - Hero image for homepage (coastal Grenada cityscape)
2. **digital-strategy.jpg** - Image for Vision & Strategy section (device/technology theme)

## Image Requirements

- **Format:** JPG or PNG
- **Size:** Optimized for web (< 500KB each)
- **Dimensions:** 
  - grenada-coastal.jpg: 1920x500px (recommended)
  - digital-strategy.jpg: 800x600px (recommended)

## Placeholder Images

Until you add your images, the site will show broken image placeholders. The functionality will work, but images won't display.

## Sources

You can use:
- Official Government of Grenada photos
- Stock photos from Unsplash/Pexels (free license)
- Custom photography

Make sure you have proper rights to use any images.
